## ğŸ“– Project Overview
**ISS Sentinel** is a real-time computer vision system designed to automatically detect and classify **7 critical safety assets** inside the International Space Station (ISS).

Trained entirely on synthetic data generated by the **Falcon (Duality AI)** platform, the model overcomes challenges like occlusion and low-light environments. To demonstrate real-world utility, the model is deployed via a **Flask Web Dashboard**, allowing astronauts to upload inspection images and receive instant safety audits.

**Target Objects:**
`OxygenTank` | `NitrogenTank` | `FirstAidBox` | `FireExtinguisher` | `SafetySwitchPanel` | `FireAlarm` | `EmergencyPhone`
